### 课程建言

1. 不同老师的讲课风格差异较大。我当时的老师全程手写板书，没有课件。文件夹中的课件是其他老师的。

2. 临近期末的习题课上会练习与考试题型相似的题型，注意参加。

3. 3系的要求是线性代数，高代作为更高要求的课程，是可以平行替代的。
